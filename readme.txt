Master Contact Form 7 - Styler for Elementor will give you full independence for Customizing Contact Form 7

== Description ==

Master Contact Form 7 - Styler for Elementor is an Addons for [Elementor](https://wordpress.org/plugins/elementor) Plugin.

This plugin is dependency of [Contact Form 7](https://wordpress.org/plugins/contact-form-7/)


>[DEMO](https://master-addons.com/contact-form-7/)  |  [Support](https://jeweltheme.com/support/forum/wordpress-plugins/master-addons/) |  [Documentation](https://master-addons.com/docs/) |  [Video Tutorials](https://www.youtube.com/watch?v=wXPEl93_UBw&list=PLqpMw0NsHXV9V6UwRniXTUkabCJtOhyIf)

== Installation ==

1. Upload the Downloaded "ma-contact-form-7.zip" Folder to the `/wp-content/plugins/` directory
2. Or, UNZIP "ma-contact-form-7.zip" folder and Upload master-addons on "wp-content/plugins/" folder via FTP or Copy and
Paste on your Local Computer
3. Activate the "Master Contact Form 7 - Styler for Elementor" Plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is there any Dependencies for using this Plugin =

Yes, Elementor Page Builder is required for using this Plugin. We've made required Plugin suggestions on Admin Dashboard.

= How can I show Contact Forms? =

First, you need to install the required plugins. Create a Contact form on Contact Form 7 Plugin.
Second, Create a Page/Post where you want to show the Widget with Elementor Editor. You'll see "Master Contact Form 7 - Styler for Elementor" Under "Master Addons" Category.
